Criar projeto React com Vite. Instalar dependências:

```bash
npm create vite@latest optfinance -- --template react
cd optfinance
npm install tailwindcss postcss autoprefixer
npm install react-router-dom
npm install recharts
npm install lucide-react
npm install date-fns
npm install clsx tailwind-merge
npx shadcn-ui@latest init
```


## 0.3 — Regras obrigatórias do Design System

Estas regras devem ser aplicadas em TODOS os componentes criados nas fases seguintes.

### Regra 1 — "No-Line Rule" (PROIBIDO bordas 1px)
- **NUNCA** usar `border border-gray-200` para separar seções ou cards
- Separação deve ser feita por mudança de cor de fundo: card (`#FFFFFF`) sobre background (`#F3F4F6`)
- Se borda for obrigatória por acessibilidade, usar "Ghost Border": `border border-outline-variant/15`

### Regra 2 — "Glass & Gradient Rule" (botões primários)
- Botão primário: gradiente vertical `from-primary to-primary-container`
- Exemplo de classe: `bg-gradient-to-t from-primary to-primary-container text-white`
- Hover: shift para `primary-fixed-dim`
- Elementos flutuantes (dropdowns, tooltips): glassmorphism com `backdrop-blur-md bg-white/40`

### Regra 3 — Sombras tintadas (NUNCA sombra preta pura)
- Sombras sempre tintadas com a cor do surface
- Usar: `shadow-sm` para cards, `shadow-lg shadow-primary-container/20` para CTAs
- Blur: 24–40px, Opacidade: 4–8%

### Regra 4 — Tipografia editorial
- Headlines e títulos: `font-bold tracking-tight` (letter-spacing -0.02em)
- Labels e badges: `uppercase tracking-widest text-xs`
- Body em leitura longa: `text-[#374151]`

### Regra 5 — Tabelas "Precision Grid"
- Header: `bg-surface-container-low` com labels em uppercase bold
- Linhas: **sem linhas horizontais ou verticais** (sem `divide-y`)
- Hover de linha: `hover:bg-row-hover` (laranja suavíssimo `#FFF7ED`)
- Status badges: `rounded-full` com fundo 10% opacity do accent + texto accent

## 0.6 — Componentes base reutilizáveis

Criar em `src/components/shared/`:

### `DataTable.jsx`
- Props: `columns` (`{key, label, render}`), `data`, `onRowClick`, `selectable`, `loading`, `emptyMessage`
- **Sem linhas divisórias** — seguir Regra 5
- Hover: `hover:bg-row-hover` (`#FFF7ED`)
- Loading: `<LoadingState />`; Empty: `<EmptyState />`

### `Pagination.jsx`
- Props: `total`, `page`, `perPage`, `onPageChange`, `onPerPageChange`
- Texto: "Página X de Y | Z registros por página"
- Botões: Primeira, Anterior, [números], Próxima, Última

### `FilterBar.jsx`
- Props: `filters` (array), `values`, `onChange`, `onClear`
- Tipos: `dateRange`, `text`, `select`, `multiselect`
- Chips ativos com "×" individual
- Botão "Limpar filtros"
- Filtros `advanced: true` ficam atrás de "+ Mais filtros"

### `SummaryCards.jsx`
- Props: `cards` (`{label, value, subtitle, delta, deltaPositive, color, icon, onClick}`)
- `delta`: variação vs período anterior (ex: "+12.4% vs anterior")
- `deltaPositive`: boolean — verde se true, vermelho se false
- Cores: `danger`, `warning`, `success`, `info`, `neutral`
- Borda esquerda colorida 4px

### `PageHeader.jsx`
- Props: `title`, `subtitle`, `breadcrumb` (array), `actions`
- Breadcrumb: "Vendas & Contratos > #0038 — AMT Engenharia" (links clicáveis exceto último)
- Ações à direita com gradiente nos botões primários

### `SlidePanel.jsx`
- Props: `open`, `onClose`, `title`, `subtitle`, `children`, `footer`
- 480px de largura, desliza da direita
- Header escuro (`inverse-surface`) com título branco
- Footer com botões "Fechar" + "Editar" (quando aplicável)
- Em mobile: largura total

### `StatusBadge.jsx`
- `rounded-full px-3 py-0.5 text-xs font-semibold uppercase tracking-wide`
- Fundo 10% opacity do accent, texto com opacity 100%

Mapeamento de status → cor:

| Status | Cor base |
|---|---|
| paga / ativo / aprovado / confirmado | verde `#2E7D32` |
| vencida / inadimplente / cancelada / perdida | vermelho `#C62828` |
| em-aberto / prevista / projeção | amarelo `#F57F17` |
| pagamento-parcial / médio | laranja `#F97316` |
| quitado / recebido | azul `#006398` |
| encerrada / inativa | cinza `#6B7280` |
| projetado / transferência | azul claro `#0288D1` |
| acao-critica | vermelho bold |

### `EmptyState.jsx` — ícone `InboxIcon` + "Nenhum registro encontrado" + mensagem opcional
### `ErrorState.jsx` — ícone `AlertCircle` vermelho + mensagem + "Tentar novamente"
### `LoadingState.jsx` — spinner `animate-spin` + "Carregando..."
### `NoPermissionState.jsx` — ícone `Lock` + mensagem + link Dashboard
