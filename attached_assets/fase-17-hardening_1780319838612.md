- [ ] Nenhuma separação usa `border border-gray-200` — usar mudança de cor de fundo

- [ ] Labels e badges com `uppercase tracking-widest text-xs`

Verificar que os alertas do Painel de Alertas Operacionais são calculados corretamente:

- [ ] "Vendas com suspeita de duplicidade" = count de vendas com badge `⚠️ POSSÍVEL DUPLICIDADE`

- [ ] "Parcelas pagas sem NF anexada" = count de parcelas com `situacao = 'paga'` e `nfEmitida = false`
- [ ] "Comissões com compensação pendente (estorno)" = count de comissões com status especial de estorno
- [ ] "Comissões aguardando NF do vendedor" = count com `status = 'aguardando-nf'`
- [ ] "Comissões prontas e não pagas" = count com `status = 'pronta'`
