## 0.5.2 — AuthContext com suporte a login

Atualizar `src/context/AuthContext.jsx` para incluir:

```js
const [autenticado, setAutenticado] = useState(false)

function login(email, senha) {
  // Mock: aceitar qualquer credencial
  setAutenticado(true)
  setUsuario({ usuario: email.split('@')[0], email, perfil: 'admin', cargo: 'Finance Director' })
}

function logout() {
  setAutenticado(false)
  setUsuario(null)
}
```

Exportar `autenticado`, `login`, `logout`.

O `PrivateRoute` deve verificar `autenticado` antes de verificar permissões. Se não autenticado: redirecionar para `/login`.

Em desenvolvimento (`import.meta.env.DEV`): manter o seletor de perfil na topbar para facilitar testes.
